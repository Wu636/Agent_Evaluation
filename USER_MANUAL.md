# 工作流智能体评测系统 - 用户使用手册

## 1. 系统简介

本系统利用大语言模型（LLM）的深度分析能力，对您的智能体对话记录进行全方位体检。系统基于**5 大维度、21 个子维度**的专业评分标准，提供客观的量化评分和具体的修改建议。

### 适用场景
*   **课程开发**：验证设计的 AI 助教是否按照预定流程进行教学
*   **质量监控**：批量评估学生与 AI 的对话质量
*   **迭代优化**：根据生成的改进建议优化 Prompt 设计

## 2. 快速开始

1.  **打开网址**：直接访问系统的在线链接（由管理员提供）。
2.  **准备数据**：
    *   **教师指导文档**：课程设计文档、Prompt 文档（支持 .doc, .docx, .md, 或直接粘贴文本）。
    *   **对话记录**：学生与 AI 的实际对话内容（支持 .json, .txt, 或直接粘贴文本）。
3.  **开始评估**：上传或输入上述两份资料，点击"开始智能评估"。
4.  **查看报告**：等待约 1-2 分钟，即可查看详细的评估雷达图和改进建议。

## 3. 核心功能详解

### 3.1 输入模式：混合文档支持

系统提供了灵活且强大的文档输入功能，支持主文档与参考资料的混合输入。

*   **教师指导文档（必填）**：
    *   **上传模式**：点击上传区域，选择本地的 Word (.doc/docx) 或 Markdown (.md) 文件。
    *   **文本模式**：直接点击 "Paste Text" 或切换按钮，粘贴您的 prompt 或教学大纲内容。
    *   这是评测的核心依据，必须提供。

*   **参考文档（可选）**：
    *   点击 **"参考文档"** 标题展开上传区域（默认收起）。
    *   可上传额外的能力训练文档、知识库片段或补充资料。
    *   系统会自动将其内容合并到评测上下文中，作为辅助参考。

> **提示**：支持混合输入场景，例如：教师文档上传 Word 文件，同时补充一份 TXT 格式的参考资料。

### 3.2 自动评估流程

点击"开始智能评估"后，系统会自动执行以下步骤：
1.  **文件解析**：提取文档中的关键信息。
2.  **并发分析**：将评估任务拆分为 21 个子任务，并发调用 LLM 进行独立评分。
3.  **结果聚合**：汇总所有评分，计算加权总分，生成五维雷达图。

**耗时说明**：通常耗时 1-3 分钟，视文档长度而定。进度条会实时显示当前评估状态。

### 3.3 评估报告解读

评估完成后，您将看到一份包含以下部分的详细报告：

*   **五维雷达图**：直观展示智能体在"目标达成"、"流程遵循"、"交互体验"、"幻觉控制"、"教学策略"五个方面的强弱项。
*   **总分**：100 分制综合评分。
*   **优先改进建议**：系统自动提取最严重的 Top 5 问题，并给出具体可执行的修改建议。
*   **维度详情**：
    *   点击右侧导航栏或滚动查看每个维度的详细分析。
    *   每个子维度（如"事实正确性"）都有 1-5 分的评分和详细的扣分理由。
    *   **严重问题**会以红色高亮显示。
*   **Markdown 导出**：点击底部"导出完整报告 (MD)"，可将报告下载到本地。

### 3.4 云端存储与历史记录

为了方便管理您的评测数据，建议**登录**系统使用。

*   **登录方式**：目前支持邮箱/密码注册登录，或 Github 快捷登录（视配置而定）。
*   **数据保存**：评估完成后，报告会自动保存到云端数据库。
*   **查看历史**：
    *   点击右上角"历史记录"图标。
    *   列表显示所有过往评测，包含文档名称、总分和时间。
    *   点击列表项可重新加载完整的评估报告。
*   **游客模式**：如果不登录，历史记录仅保存在浏览器本地，清除缓存会丢失。

### 3.5 分享与探索

您可以将评估报告分享给团队成员或公开展示。

*   **生成分享链接**：
    *   在历史记录列表中，点击 **"分享图标"**（🔗）。
    *   系统会生成一个唯一的访问链接，点击复制按钮即可发送给他人。
    *   被分享者无需登录即可查看该报告。
    
*   **公开/私有切换**：
    *   点击 **"锁形图标"**（🔒/🔓）可切换报告状态。
    *   **私有（默认）**：只有您自己和拥有分享链接的人可见。
    *   **公开**：报告将出现在"探索广场"，所有人可见。

*   **探索广场**：
    *   点击导航栏右上角的 **"探索"** 按钮（或罗盘图标）。
    *   这里展示了社区用户公开的优秀智能体评测案例，您可以浏览学习他人的 Prompt 设计与评估结果。

### 3.6 协作与通知

系统内置了团队协作功能，方便您与同事就评测结果进行讨论。

*   **评论功能**：
    *   在评测报告底部，您可以发表评论，记录想法或提出建议。
    *   支持对具体的评测维度或整体结果进行讨论。

*   **@提及 (Mention)**：
    *   输入 `@` 符号，系统会自动弹出用户列表。
    *   选择特定的同事进行“提及”，对方将会收到通知。
    *   支持直接输入邮箱来提及未在列表中显示的用户。

*   **通知中心**：
    *   点击右上角的 **铃铛图标** 查看消息通知。
    *   当有人回复您的评论或 `@` 您时，这里会有红点提示。
    *   点击通知可直接跳转到对应的评测报告和评论位置。

## 4. 常见问题 (FAQ)

**Q: 评估过程中进度条卡住怎么办？**
A: 系统采用并发请求，偶尔可能因网络波动等待响应。请耐心等待，如果超过 5 分钟无进展，建议刷新页面重试。此次操作不会消耗您的实际 API 额度（如果计费是按 Token 算的）。

**Q: 上传的 Word 文档格式乱了有影响吗？**
A: 不影响。系统只提取文本内容进行语义分析，格式（字体、颜色等）会被忽略。

**Q: 对话记录需要什么格式？**
A: 推荐使用 JSON 格式（包含 `role` 和 `content`），或者简单的文本对话格式（如 "User: ..., Agent: ..."）。只要 LLM 能区分出谁在说话即可。

**Q: 我的数据安全吗？**
A: 
*   **未登录**：数据完全在本地处理，评估结束后即销毁（除浏览器缓存外）。
*   **已登录**：数据加密存储在云端数据库，仅您自己可见（除非您主动设置为公开）。
*   LLM 评估时数据会发送给模型提供商（如 OpenAI），但遵循其隐私政策用于推理，不用于训练模型。

**Q: 为什么同一个对话测两次分数不一样？**
A: LLM 评估具有微小的随机性，这是正常的。但只要 Prompt 质量稳定，分数波动通常在 5 分以内。
