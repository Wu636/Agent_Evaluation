# 🤖 LLM 智能体评测系统

## 用户使用手册

一个专业的 AI 教学智能体质量评估工具，帮助您快速、全面地评估 LLM 智能体的教学效果。

---

## ✨ 核心功能

### 1. 多维度智能评测

系统从 **5 个一级维度 + 21 个二级维度** 全面评估智能体表现：

| 一级维度 | 分值 | 二级维度 |
|---------|------|----------|
| 🎯 目标达成度 | 20分 | 知识点覆盖率、能力覆盖率 |
| 📋 流程遵循度 | 20分 | 环节准入条件、内部顺序、全局流转、准出检查、非线性跳转 |
| 💬 交互体验性 | 20分 | 人设语言风格、表达自然度、上下文衔接、循环僵局、回复长度 |
| 🔍 幻觉与边界 | 20分 | 事实正确性、逻辑自洽性、未知承认、安全围栏、干扰抵抗 |
| 📐 教学策略 | 20分(加分) | 启发式提问、正向激励、纠错引导、深度追问 |

### 2. 流式实时评估

- ⚡ **流式响应**：后端逐 token 输出，避免超时
- 📊 逐维度展示评估进度
- 🚀 5 路并发，速度提升显著

### 3. 报告交互优化

- 📍 侧边栏折叠式子维度列表
- 🎯 点击子维度可精确滚动到对应位置
- 📋 严重问题按维度分组折叠展示

### 4. 历史记录管理

- 📁 自动保存每次评估结果
- 🔍 随时查看历史报告
- 🗑️ 支持删除不需要的记录

### 5. 文件持久化

- 💾 上传的文件自动保存到浏览器
- 🔄 切换页面后文件仍在
- 🧪 方便用不同模型重复测试

## 📖 使用指南

### 第一步：准备文件

您需要准备两个文件：

1. **教师指导文档**（`.doc`、`.docx` 或 `.md` 格式）
   - 包含教学目标、教学流程、评分标准等

2. **对话记录**（`.json` 或 `.txt` 格式）
   - 智能体与学生的完整对话历史

### 第二步：配置设置

点击右上角 **设置** 按钮，配置：

- **API 密钥**：您的 LLM 服务密钥
- **API 地址**：LLM 服务端点
- **评测模型**：选择用于评测的模型

> 💡 **支持的模型**：GPT-4o、Claude Sonnet、DeepSeek 等

### 第三步：上传文件

1. 将教师指导文档拖拽到第一个上传框
2. 将对话记录拖拽到第二个上传框
3. 两个文件都上传后，点击 **开始评估**

### 第四步：查看报告

评估完成后，您将看到：

- 📈 **总体评分**：综合得分和等级
- 🕸️ **雷达图**：各维度得分可视化
- 📝 **子维度详情**：每个二级维度的具体评价和证据引用
- ⚠️ **严重问题汇总**：按维度分组的问题列表
- 💡 **优先改进建议**：Top 5 改进建议

## 🔧 高级功能

### 1. 切换模型测试

- 完成一次评估后，返回首页
- 文件会自动保留
- 在设置中切换不同模型
- 重新点击"开始评估"即可对比

### 2. 快速导航

- 右侧边栏显示所有子维度得分
- 点击任意子维度可跳转到对应详情

### 3. 导出报告

- 点击"导出完整报告 (MD)"
- 下载 Markdown 格式的完整评测报告

## 📊 评分等级说明

| 等级 | 分数范围 | 说明 |
|------|----------|------|
| 🏆 优秀 | 90-100 分 | 表现卓越，全面达标 |
| 👍 良好 | 75-89 分 | 表现良好，部分可优化 |
| ✅ 合格 | 60-74 分 | 基本达标，需要改进 |
| ⚠️ 不合格 | 0-59 分 | 未达标，需要重大改进 |

## ❓ 常见问题

### Q: 评估需要多长时间？

通常 2-4 分钟，取决于对话记录长度和 LLM 响应速度。

### Q: 支持哪些文件格式？

- 教师文档：`.doc`、`.docx`、`.md`
- 对话记录：`.json`、`.txt`

### Q: 历史记录保存在哪里？

历史记录保存在您的浏览器本地存储 (IndexedDB) 中，清除浏览器数据会丢失。
