# 🤖 LLM 智能体评测系统

## 用户使用手册

一个专业的 AI 教学智能体质量评估工具，帮助您快速、全面地评估 LLM 智能体的教学效果。

---

## ✨ 核心功能

### 1. 多维度智能评测

系统从 **6 个专业维度** 全面评估智能体表现：

| 维度 | 权重 | 说明 |
|------|------|------|
| 🎯 目标达成度 | 40% | 教学目标是否完成（一票否决项） |
| 📐 策略引导力 | 20% | 引导学生思考的策略质量 |
| 📋 流程遵循度 | 15% | 是否按照教学流程执行 |
| 💬 交互体验感 | 10% | 对话自然度和用户体验 |
| 🔍 幻觉控制力 | 10% | 回答准确性，避免编造内容 |
| 🛡️ 异常处理力 | 5% | 处理异常输入的能力 |

### 2. 实时流式评估

- ⚡ 实时显示评估进度
- 📊 逐维度展示评估结果
- ⏱️ 避免长时间等待超时

### 3. 历史记录管理

- 📁 自动保存每次评估结果
- 🔍 随时查看历史报告
- 🗑️ 支持删除不需要的记录

### 4. 文件持久化

- 💾 上传的文件自动保存
- 🔄 切换页面后文件仍在
- 🧪 方便用不同模型重复测试

---

## 📖 使用指南

### 第一步：准备文件

您需要准备两个文件：

1. **教师指导文档**（`.docx` 或 `.md` 格式）
   - 包含教学目标、教学流程、评分标准等

2. **对话记录**（`.json` 或 `.txt` 格式）
   - 智能体与学生的完整对话历史

### 第二步：配置设置

点击右上角 **设置** 按钮，配置：

- **API 密钥**：您的 LLM 服务密钥
- **API 地址**：LLM 服务端点
- **评测模型**：选择用于评测的模型

> 💡 **支持的模型**：GPT-4、GPT-4o、Claude Sonnet、DeepSeek 等

### 第三步：上传文件

1. 将教师指导文档拖拽到第一个上传框
2. 将对话记录拖拽到第二个上传框
3. 两个文件都上传后，点击 **开始评估**

### 第四步：查看报告

评估完成后，您将看到：

- 📈 **总体评分**：综合得分和等级
- 🕸️ **雷达图**：各维度得分可视化
- 📝 **详细分析**：每个维度的具体评价
- ⚠️ **关键问题**：需要改进的问题列表
- 💡 **优化建议**：具体的改进建议

## 🔧 高级功能

### 1. 切换模型测试

- 完成一次评估后，返回首页
- 文件会自动保留
- 在设置中切换不同模型
- 重新点击"开始评估"即可对比

### 2. 查看历史记录

- 点击右上角 **历史记录** 按钮
- 浏览所有历史评估
- 点击眼睛图标查看详细报告
- 点击垃圾桶图标删除记录

### 3. 清空已上传文件

- 首页右上角会显示 **清空文件** 按钮
- 点击可清除所有已上传文件
- 开始全新的评估

## 📊 评分等级说明

| 等级 | 分数范围 | 说明 |
|------|----------|------|
| 🏆 优秀 | 90-100 分 | 表现卓越，全面达标 |
| 👍 良好 | 75-89 分 | 表现良好，部分可优化 |
| ✅ 合格 | 60-74 分 | 基本达标，需要改进 |
| ⚠️ 不合格 | 0-59 分 | 未达标，需要重大改进 |
| 🚫 一票否决 | - | 目标达成度低于 60 分时触发 |

## ❓ 常见问题

### Q: 评估需要多长时间？

通常 1-3 分钟，取决于对话记录长度和选择的模型。

### Q: 支持哪些文件格式？

- 教师文档：`.docx`、`.md`
- 对话记录：`.json`、`.txt`

### Q: 历史记录保存在哪里？

历史记录保存在您的浏览器本地存储中，清除浏览器数据会丢失。

### Q: 可以同时评估多个文件吗？

目前每次只能评估一组文件，但您可以利用历史记录功能保存多次评估结果。
